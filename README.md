# Hospital Operations Management Information System (HOMIS)

A modern healthcare MIS platform for hospital operations, analytics, and reporting.

## Modules
- Patient Management
- Pharmacy & Inventory
- Revenue Tracking
- Staff Performance
- Bed Management
- Lab Tracking

## Tech Stack (Planned)
- Backend: FastAPI / Django
- Frontend: React
- Database: PostgreSQL
- Deployment: Docker + Cloud (Huawei/AWS)
